# portfolio_-website
 Mennas Fransiska

A clean and elegant personal portfolio website showcasing web design and development skills.

 Features

- Responsive Design: Fully responsive layout that works on desktop and mobile devices
- Multiple Pages: Home, About, Skills, Projects, and Contact sections
- : Split-screen hero layout with soft pink color scheme
- Professional Typography: Using Inter font family for clean readability

 Technologies Used

- HTML
- CSS


 CSS Requirements Implemented

1. Multiple heading sizes (h1, h2, h3)
2. Images with margin and padding
3. Navigation links and call-to-action buttons
4. Paragraphs with dotted borders
5. Multiple div containers for layout structure
6. Custom background colors throughout
